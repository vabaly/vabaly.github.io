# cUrl 格式化辅助

因为 Hoppscotch (opens new window) 的 cUrl 导入功能对于某些 cUrl 存在以下 Bug：

  1. cUrl 的 url 字符串包含 " 或 '，导致最后一个查询参数的值会带上 " 或 '；
  2. cUrl 的 Headers 中包含 Cookie 字段，Hoppscotch 不会识别，得转成小写的 cookie 使其识别。

所以这些 cUrl 在交给 Hoppscotch 处理前要先格式化一下。
